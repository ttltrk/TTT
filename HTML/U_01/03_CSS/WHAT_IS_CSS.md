### basics

```
Cascading Style Sheets
CSS defines how HTML looks like
HTML is content
CSS is style
```

### CSS code inside an HTML tag

```html
<p style="front-size: 120%">
```

### CSS code inside an HTML document

```html
<style>
  p {
    front-size: 120%
  }
</style>
```

### CSS code in external file

```css
p {
  front-size: 120%
}
```
