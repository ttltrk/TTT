#### UNION_VS_UNION_ALL

```
UNION ALL command is equal to UNION command, except that UNION ALL select all values. It will not
remove duplicate rows instead it will retrieve all rows from all tables.
```

```sql
select * from temp1

UNION

select * from temp2

>>>

>>>
```

```sql
select * from temp1

UNION ALL

select * from temp2

>>>

>>>
```
