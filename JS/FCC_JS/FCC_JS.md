
#### 1 - comments

```js
// This is an in-line comment.
/* test */
```

#### 2 - declare variables

```js

```

#### 3 - assignment operator

```js
var a;

```

#### 4 - assigne variable to another

```js
var a;
a = 7;
var b;
b = a;
```

#### 5 - initializing variables

```js
var a = 9;
```

#### 6 declare string variables
