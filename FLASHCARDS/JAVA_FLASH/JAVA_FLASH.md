#### 01_COMMENTS

##### 01.01 - Single line comments

```java
public class Main {
  public static void main(String[] args) {
    // This is a comment
    System.out.println("Hello World");
  }
}

>>>
Hello World
>>>
```

##### 01.02 - Multi line comments

```java
public class Main {
  public static void main(String[] args) {
    /* The code below will print the words Hello World
    to the screen, and it is amazing */
    System.out.println("Hello World");
  }
}

>>>
Hello World
>>>
```
