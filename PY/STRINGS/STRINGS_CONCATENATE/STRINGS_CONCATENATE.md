#### BASICS

To concatenate, or combine, two strings you can use the ```+``` operator.

```py
a = "Hello"
b = "World"
c = a + b

print(c)

>>>
HelloWorld
>>>
```

To add a space between them, add a ```" "```

```py
a = "Hello"
b = "World"
c = a + " " + b

print(c)

>>>
Hello World
>>>
```
