<h3 id='slc'>Single Line Comments</h3>

```py
#This is a comment
print("Hello, World!")

>>>
Hello, World!
>>>
```

```py
print("Hello, World!") #This is a comment

>>>
Hello, World!
>>>
```

```py
#print("Hello, World!")
print("Cheers, Mate!")

>>>
Cheers, Mate!
>>>
```
