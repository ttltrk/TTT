<h3 id='append'>append()</h3>

```
Adds an element at the end of the list
```

```py
fruits = ['apple', 'banana', 'cherry']
fruits.append("orange")

print(fruits)

>>>
['apple', 'banana', 'cherry', 'orange']
>>>
```
